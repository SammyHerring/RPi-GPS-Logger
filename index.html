Welcome to Sport Trax's RPi GPS Logger

A Raspberry Pi-based GPS logger used for running; cycling and/or other leisure activities using the Python library as well as other dependencies.

The project will be housed in custom casing using the following components:

Powered by Raspberry Pi 2 (running Jessie), programmed using Python Adafruit Powerboost 500c Lipo Charger 2.8" TFT Touchscreen for RPi Adafruit Weatherproof Metal On-Off Switch with LED (which will act as power indicator for the board) GlobalSat BU-353-S3 USB GPS Reciever (USB Port)
